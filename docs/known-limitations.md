# Known Limitations

This project is intentionally small.

## Not Production Ready

This is a local portfolio lab, not a production cloud service.

It does not include:

- production authentication
- real customer data
- real secrets
- autoscaling
- Kubernetes
- cloud deployment
- production monitoring
- alerting
- tracing
- incident management tooling

## Why These Limits Are Good

The project is meant to prove beginner-to-junior support skills clearly.

Adding Kubernetes, microservices, complex auth, or paid cloud services too early would make the project harder to explain and easier to break without adding much value for the target roles.

## What a Production Version Would Need

A production version would need:

- secure secret management
- authentication and authorization
- structured deployment environments
- monitored logs and metrics
- alerting
- backup and rollback strategy
- real dependency checks
- security review
- infrastructure-as-code

Those are useful later, but not required for this portfolio v1.
