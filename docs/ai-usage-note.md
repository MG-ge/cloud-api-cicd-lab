# AI Usage Note

AI was used as a mentor and coding assistant for this project.

The owner is responsible for understanding and verifying:

- what the API endpoints do
- what the tests prove
- what Docker does
- what CI/CD does
- how support cases were investigated
- what the known limitations are

The project should not be presented as production experience.

It should be presented as a hands-on learning and portfolio project for junior SaaS, application support, cloud support, and integration-adjacent roles.
